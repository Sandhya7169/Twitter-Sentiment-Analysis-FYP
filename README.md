# Final Year Project: Sentiment Analysis on Twitter Data

This project aims to perform sentiment analysis on tweets using machine learning. It includes a Flask web application that allows users to input text and get sentiment prediction (positive, negative, neutral).

## Features
- Data preprocessing and cleaning
- Model training using Logistic Regression
- Web interface for real-time prediction
- API endpoint for predictions

## Installation
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the Flask app: `python app.py`
4. Open browser at `http://localhost:5000`

## Dataset
The dataset used is a sample of tweets from Kaggle. It is stored in `data/tweets.csv`.

## Model
The trained model is saved as `sentiment_model.pkl`. You can retrain the model using `model.py`.
